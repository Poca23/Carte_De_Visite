# 🃏 Carte de visite — CND · Web Is Yours

Éditeur de carte de visite interactive en HTML/CSS/JS vanilla.  
Personnalisez chaque élément, prévisualisez en temps réel et imprimez.

## 🚀 Lancement

Ouvrir `index.html` dans un navigateur. Aucune installation requise.

## 📁 Structure

```
├── index.html
├── css/
│ ├── main.css
│ ├── base.css
│ ├── layout.css
│ ├── panel.css
│ ├── card.css
│ └── print.css
└── js/
├── main.js
├── render.js
├── history.js
├── drag.js
└── media.js
```

## ✏️ Fonctionnalités

- Édition en temps réel (textes, couleurs, dégradés)
- Upload de logos (recto / verso)
- QR Code auto-généré (qrcodejs CDN)
- Drag & drop des éléments
- Historique Annuler / Rétablir
- Sauvegarde automatique (localStorage)
- Impression via le navigateur
